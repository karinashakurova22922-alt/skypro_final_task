Git

  • git init - инициализирует новый пустой Git-репозиторий в текущей директории.
  • git clone - копирует существующий репозиторий из удаленного источника (например, GitHub) на твой компьютер.
  • git status - показывает текущий статус рабочего каталога и индекса, то есть какие изменения подготовлены для коммита, какие файлы не отслеживаются и т.д.
  • git push - отправляет коммиты из локального репозитория в удаленный репозиторий.
  • git pull - обновляет локальный репозиторий, подтягивая изменения из удаленного репозитория
  • git add - добавляет изменения в индекс (стейджинг), подготавливая их к последующему коммиту.
  • git commit - фиксирует изменения, добавленные через git add, в репозиторий.
